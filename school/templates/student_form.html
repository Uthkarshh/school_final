Upload a CSV file with student information for bulk import

Student Management

This form is used to add or update student personal and contact information. Each student needs a unique PEN (Personal Education Number) which will be used to link their fee records, class details, and transport information.

CSV Import Format

Your CSV file should have the following columns:

pen_num,admission_number,aadhar_number,student_name,father_name,mother_name,gender,date_of_birth,date_of_joining,contact_number,village

Student Record Tips

  • Ensure that the PEN number is unique for each student
  • The Aadhar number should be exactly 12 digits
  • Dates should be in YYYY-MM-DD format
  • After adding a student, create corresponding class details record
  • Update contact information whenever it changes to maintain accurate records
